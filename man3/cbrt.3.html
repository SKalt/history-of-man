NAME

cbrt, cbrtf, cbrtl - cube root function  

SYNOPSIS

#include <math.h>

double cbrt(double x);


float cbrtf(float x);


long double cbrtl(long double x);

Link with -lm.

Feature Test Macro Requirements for glibc (see feature_test_macros(7)):

cbrt(), cbrtf(), cbrtl(): _BSD_SOURCE || _SVID_SOURCE || _XOPEN_SOURCE >= 500 || _ISOC99_SOURCE; or cc -std=c99  

DESCRIPTION

The cbrt() function returns the (real) cube root of x. This function cannot fail; every representable real value has a representable real cube root.  

CONFORMING TO

C99  

SEE ALSO

pow(3), sqrt(3)
