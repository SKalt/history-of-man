NAME

fabs, fabsf, fabsl - absolute value of floating-point number  

SYNOPSIS

#include <math.h>

double fabs(double x);


float fabsf(float x);


long double fabsl(long double x);

Link with -lm.  

DESCRIPTION

The fabs functions return the absolute value of the floating-point number x.  

ERRORS

No errors can occur.  

CONFORMING TO

The fabs() function conforms to SVID 3, POSIX, 4.3BSD, ISO 9899. The other functions are from C99.
