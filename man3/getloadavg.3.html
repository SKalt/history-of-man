NAME

getloadavg - get system load averages  

SYNOPSIS

Fd #define _BSD_SOURCE Fd #include <stdlib.h> Ft int Fn getloadavg double loadavg[] int nelem  

DESCRIPTION

The Fn getloadavg function returns the number of processes in the system run queue averaged over various periods of time. Up to Fa nelem samples are retrieved and assigned to successive elements of Fa loadavg Ns Bq . The system imposes a maximum of 3 samples, representing averages over the last 1, 5, and 15 minutes, respectively.  

RETURN VALUE

If the load average was unobtainable, -1 is returned; otherwise, the number of samples actually retrieved is returned.  

HISTORY

The Fn getloadavg function appeared in BSD 4.3 Reno This function is available in glibc since version 2.2.  

CONFORMING TO

Not in POSIX.1-2001. Present on the BSDs and Solaris.
